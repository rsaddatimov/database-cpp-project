# Key-value (data, event) database

## Description
This is my project where I implemented key-value database, which stores (data, event) and can perform some operations with conditions and predicates.

## What database can
- Storing (date, event) pairs.

## How to build
```
mkdir build
cd build
cmake ..
make
./database
```

## Stack
- C++ 17
- STL, Algorithms
- CMake
